## 💡 فكرة المشروع

تطبيق لإدارة المهام اليومية بشكل بسيط وفعّال. يسمح لك بـ:

✅ إضافة المهام  
✅ تحديد المهام كمكتملة  
❌ حذف المهام  
💾 حفظ المهام تلقائيًا باستخدام Local Storage  



## ⚙️ التقنيات المستخدمة

- ⚛️ React
- 💙 TypeScript
- ⚡ Vite
- 🖼️ CSS with Gradient Styling
- 🧠 LocalStorage
